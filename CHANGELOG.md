# Changelog  

All notable changes to this project will be documented in this file.

## [0.1.2] - 2024-04-22  

### Added  
- Populated Help tabs with links to wiki docs.  

### Changed  
- Upload data label and tooltip.

### Added

- About page loads from a .md.

- Redundant analyse datatable UI.
